# VTS Universe Sdn Bhd — Website

A marketing/portfolio site for VTS Universe Sdn Bhd, a geospatial solutions company offering aerial LiDAR, terrestrial surveys, and GIS analytics.

## Stack

- **Framework**: TanStack Start (React 19 + SSR)
- **Routing**: TanStack Router (file-based, `src/routes/`)
- **Styling**: Tailwind CSS v4 + shadcn/ui (Radix UI)
- **Package manager**: Bun
- **Build tool**: Vite 8

## Running the project

```bash
bun install        # install dependencies
bun run dev        # start dev server on http://localhost:5000
```

The **"Start application"** workflow runs `bun run dev` and serves the app on port 5000.

## Project structure

```
src/
  routes/           # File-based routes (TanStack Router)
    __root.tsx      # App shell / root layout
    index.tsx       # Home page (/)
    about.tsx       # /about
    services.tsx    # /services
    experiences.tsx # /experiences
    contact.tsx     # /contact
  components/       # Shared UI components
  hooks/            # Custom React hooks
  lib/              # Utilities
  styles.css        # Global styles (Tailwind)
  server.ts         # SSR error wrapper
  start.ts          # TanStack Start entry point
```

## Notes

- `src/routeTree.gen.ts` is auto-generated by TanStack Router — do not edit by hand.
- The project was originally built on [Lovable](https://lovable.dev). The vite config was adapted for Replit (port 5000, `allowedHosts: true`).
